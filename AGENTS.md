# 감성 분석 서비스 프로젝트 에이전트 지침

이 프로젝트는 HTML, CSS, JavaScript 기반 감성 분석 서비스를 구현하기 위한 프로젝트이다.

AI 코딩 에이전트는 반드시 docs 폴더 내부 문서를 기준으로 구현한다.

---

# 프로젝트 목표

사용자가 문장을 입력하면 OpenAI API를 통해 감성을 분석하고 아래 정보를 출력한다.

- 긍정 / 부정 / 중립
- 신뢰도 백분율
- 분석 이유
- 오류 메시지

---

# 기술 스택

- Front-end: HTML, CSS, JavaScript
- Back-end: Node.js + Express
- AI: OpenAI API
- DB: Supabase
- Deploy: Vercel

---
